OOP project 
Brief explanation of each class and its responsibility :

Class person : This is the base class it collects general information related to all individuals such as (Name, phone number, email) serves as a blueprint for more specific roles.

Class client: This is a derived class from the parent class its responsibility is related to the users specific data such as payment methods ,booking history.
Service providers 
Class service: it’s a base class of all services the company provides .
Class catering: it is a class that inherits from the bass class (service) which hold anything related to food that the company can provide.

Class decoration: it is a class that inherits from the bass class (service) which hold anything related to decoration that the company can provide.

Class entertainment: it is a class that inherits from the bass class (service) which hold anything related to the activities that the company can provide.

Class photography: it is a class that inherits from the bass class (service) which hold anything related to photography that the company can provide.

Class event :class that specifies the type of event such as (birthday party, graduation party etc..)

Class venue: class that specifies the location of the event
